# newproject-3

Course project for **מבוא לבסיסי נתונים ובינה עסקית** (Introduction to Databases and Business Intelligence), Business Administration program.

## Overview

This repository contains starter Python code used as part of the course work. It currently includes a small sample script that demonstrates a simple greeting function.

## Project structure

```
newproject-3/
├── README.md
└── sampplecode.py    # Sample Python script
```

## Requirements

- [Python 3](https://www.python.org/downloads/) (3.8 or newer recommended)

No external packages are required for the current code.

## Getting started

1. Clone the repository:

   ```bash
   git clone https://github.com/shaharlem/newproject-3.git
   cd newproject-3
   ```

2. Run the sample script:

   ```bash
   python sampplecode.py
   ```

   Expected output:

   ```
   שלום, עולם!
   ```

## Usage

`sampplecode.py` defines a `greet(name)` function that prints a Hebrew greeting. You can import and reuse it in other scripts:

```python
from sampplecode import greet

greet("שם")
```
